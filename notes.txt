Do this after cloning

1. run 'npm i' in both client and server directory
2. the framer-motion supports only on version 6 (not fixed yet for latest versions)
3. install Animate-on-Scroll 'npm install aos --save' 'import AOS from aos'
4. install React Big Calendar 'npm install --save react-big-calendar'
5. install SwiperJS 'npm i swiper'
6. 	to run in CLIENT:
		'npm run dev'
	to run in SERVER:
		'npm start'

To run sequelize

1. run 'npm i' in server directory
2. create database 'npoc' in MySQL Workbench.
3. run 'npm start' in server
